# README

This is platform for decentralized organization. It can be used for groups of varying sizes and purposes. It relies on criteria-driven problem solving and crowdacting to organize members in a non-hierarchical way.

Specs
* Ruby 2.5.1
* Rails 5.1.4
* PostgreSQL

Config variables
* ADMIN_EMAIL
* ADMIN_PASSWORD
* SERVER_HOST
* GROUP_NAME
* GROUP_ABBREVIATION

Setup
* Start Postgres
* Run 'bundle install'
* Run 'rails db:create'
* Run 'rails db:migrate'
* Run 'rails db:seed'
* Run 'rails server'
